# Prompt 23: 关联台强搜索弹窗与跳转定位

## 目标

在关联台页面增加统一搜索入口和搜索弹窗，按 `OA / 银行流水 / 发票` 分组展示结果，并支持一键跳回关联台定位到目标记录。

## 你要做的事

- 在关联台顶部新增搜索框入口
- 删除旧的 `关联台月份 2026-03` 纯文案区域
- 保留页内月份选择器
- 新增搜索弹窗，包含：
  - 搜索输入框
  - 范围切换：全部 / OA / 流水 / 发票
  - 月份筛选
  - 项目筛选
  - 状态筛选
- 结果按三组展示：
  - OA
  - 银行流水
  - 发票
- 每条结果提供 `跳转至`
- 点击后：
  - 关闭弹窗
  - 切换月份
  - 回关联台
  - 展开到正确区域
  - 滚动并高亮到目标记录

## 约束

- 结果区不要复用三栏候选布局
- 结果项应是普通列表
- `跳转至` 一律落到关联台，不跳去成本统计或税金抵扣
- 要支持：
  - 已配对
  - 未配对
  - 已忽略
  - 已处理异常
- 已忽略 / 已处理异常 需要自动打开对应弹窗后再定位

## 建议结果项信息

OA：
- 项目名称
- 申请人 / 对方户名 / 金额
- 费用类型 / 费用内容
- 月份 / 状态 / 命中字段

银行流水：
- 对方户名 / 公司名
- 交易时间 / 金额 / 收入支出
- 支付账户 / 流水号
- 摘要 / 备注
- 月份 / 状态 / 命中字段

发票：
- 发票号码
- 销方名称 / 购方名称 / 金额
- 开票日期 / 发票类型
- 月份 / 状态 / 命中字段

## 验收标准

- 页面顶部出现统一搜索入口
- 点击后打开搜索弹窗
- 结果按 OA / 流水 / 发票 分组
- 每条结果都可跳转
- 跳转后可以切月份并定位到正确记录
- 前端测试通过
