# Prompt 18: 成本统计导出与验收

## 目标

补齐 `成本统计` 页面导出能力、文件命名规则、状态反馈和端到端回归。

## 你要做的事

- 新增当前视图导出能力
- 导出文件名带年月和层级上下文
- 补导出 loading / success / error 反馈
- 完成前后端联调与回归测试

## 约束

- 导出必须跟随当前视图层级
- 不导出与当前层级无关的数据
- 优先 `xlsx`，必要时允许 `csv`
- 不要破坏已有工作台与税金页

## 交付物

- 成本统计导出接口或前后端导出链路
- 前端导出按钮与状态反馈
- 测试与验收说明

## 验收标准

- 月份汇总层可以导出当前月份汇总
- 项目明细层可以导出当前项目明细
- 流水层可以导出当前流水详情 / 明细
- 导出文件名含 `年月 / 层级 / 项目名称（如适用）`
- 前后端测试通过

## 可直接使用的 Prompt

```md
请为“成本统计”页面补齐导出与验收。

背景：
- `成本统计` 页面已经有月份汇总、项目明细和具体流水下钻
- 现在需要增加导出能力，并把整条链路验收完整

实现要求：

1. 新增导出能力，必须跟随当前页面层级：
   - 月份汇总层导出月份汇总
   - 项目明细层导出当前项目明细
   - 流水层导出当前流水详情或流水明细
2. 导出文件名需包含：
   - 年月
   - 当前层级
   - 项目名称（如适用）
3. 前端导出过程中要有清晰反馈：
   - loading
   - success
   - error
4. 补前后端测试与验收说明。

注意：
- 不要导出无关层级数据。
- 优先保证导出结果和当前页面看到的内容一致。
```
